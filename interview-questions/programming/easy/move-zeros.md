Given an array `nums`, implement a method to move all 0's to the end of it
while maintaining the relative order of the non-zero elements.

    void moveZeros(int[] nums) {
        // ...
    }

Example inputs and expected results:

- [0, 1, 0, 2, 3] -> [1, 2, 3, 0, 0]
- [5, 0, 1] -> [5, 1, 0]
- [1, 2, 3] -> [1, 2, 3]
- [0, 0] -> [0, 0]
- [] -> []
